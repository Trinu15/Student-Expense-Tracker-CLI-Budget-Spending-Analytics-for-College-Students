# Student Expense Tracker CLI

A Python command-line application for students to track daily expenses.

Features:
- Add expenses
- View expenses
- Total spending calculation
- Highest spending category
- Category-wise report
- Delete expenses
- Monthly budget warning
- CSV file storage

How to Run:
1. Install Python 3
2. Run: python expense_tracker.py

Future Improvement:
Add charts, monthly analytics, and SQLite storage.
